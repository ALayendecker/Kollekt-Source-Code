## Getting Started

### Installation

1. Clone the repo

```sh
git clone https:://github.com/Alayendecker/Kollekt.git
```

2. Install NPM packages

```sh
npm install
```

3. Create an `.env` to hide your own mysql-database-root-password/mysql-database-config you will need to run this app.
